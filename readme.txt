== Description ==

 Cart2Cart: Volusion to WooCommerce Migration plugin is a uniquely convenient data transfer tool that can perform an accurate automated switch in the easiest way possible. This plugin doesn’t require any programming skills, but allows to move products, orders, customers, categories, reviews and other entities in no time. Everything you need to do is to establish a connection between the stores, choose the wanted entities and additional options and let Cart2Cart take care of the rest. Though the plugin itself is free to download, the migration is paid and starts from $69.

**Move the following entities from Volusion to WooCommerce:**

* Products, product images, product extra fields, product attributes;
* Categories, category images;
* Customers, customer shipping address, customer billing address;
* Orders, order products;
* Manufacturers, reviews.

*Supported WooCommerce versions:* 1.1.x - 1.6.x, 2.x
**Please note:** Cart2Cart migrates data only. Your current Volusion store design template won’t be transferred to WooCommerce. 

Cart2Cart: Volusion to WooCommerce Migration plugin gives a unique possibility to move with no efforts and difficulties. As Volusion is API based platform, you just need to provide its URL, while WooCommerce still requires Connection Bridge to connect the stores. Then, click a button of start and just follow simple Wizard instructions. 

**Pre-Migration Steps**

1. Install WooCommerce store and ensure its accessible via internet.
1. Check for FTP details for your WooCommerce shopping cart. You need them to perform migration.

== Installation ==
1. Download the plugin in a zip. archive
1. Unpack it on your computer.
1. Upload the extracted file to wp-content/plugin directory
1. Go to the plugins section of your admin panel and turn on the “Cart2Cart Volusion to WooCommerce Migration” module.
1. Insert Volusion store’s URL 
1. Connection Bridge for WooCommerce  will be uploaded automatically
1. Click on “Start Migration”, follow the simple instructions on Migration Wizard and have your data transferred in a few hours.

== Frequently Asked Questions ==
= Is Connection Bridge installation needed for Volusion switch? =
No, as Volusion is API-based solution, you don’t need to upload Connection Bridge files.

= Will clients credit card information be moved during transfer? = 
Unfortunately, service doesn’t provide migration of credit card data.
= Do you support “Preserve product image URLs” option? =
Yes, this is an additional option and it allows to transfer only the link structure of images. It means that images are not going to be shown on Volusion shop. So, you will have to transfer them from Source Store to Target one using FTP saving file names and folder structure. In case when this option is not selected, only images links will be moved and images will be displayed from your Source Store.
= Is migration of review supported by Cart2Cart?= 
Yes, Cart2Cart provides reviews transfer. However, these entities is not going to be migrated during Demo Migration, but they will appear on your Target Store after completing Full transfer.
